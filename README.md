# 🤖 AI Interview Coach (AI Engineering)

An intelligent interview preparation tool designed to simulate real-world AI/Software Engineering interviews with structured, real-time feedback.

## 🚀 Features

* **Context Management**
  Efficient session state handling in Streamlit to maintain long conversation history while reducing token usage.

* **LLM Integration**
  Integrated OpenAI and Gemini APIs to deliver:

  * Coding feedback
  * Behavioral response analysis
  * Structured evaluation

* **Real-Time Feedback**
  Get instant suggestions on logic, clarity, and communication.

## 🛠 Tech Stack

* Python
* Streamlit
* LangChain
* OpenAI API
* Gemini API

## 🎯 Use Case

Practice technical and behavioral interviews with AI-driven insights to improve performance and confidence.
